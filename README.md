# Weather App ⛅

A clean, elegant, and responsive weather web application that allows users to check current weather conditions for any location around the world.

## 🚀 Features
* **Location Search:** Easily search for any city or location.
* **Real-time Data Display:** View current temperature and weather conditions.
* **Detailed Metrics:** Shows additional weather data including:
  * Humidity levels
  * Wind speed
* **Dynamic Icons:** Visual representation of the current weather.
* **Error Handling:** Built-in "Not Found" state for invalid location entries.

## 🛠️ Technologies Used
* **HTML5:** For structuring the web application.
* **CSS3:** For styling, layout, and animations.
* **JavaScript:** For fetching API data and dynamically updating the DOM.
* **External Assets:**
  * [Google Fonts](https://fonts.google.com/) (Poppins & Roboto)
  * [FontAwesome](https://fontawesome.com/) & [Icons8](https://icons8.com/) for UI icons.

\`\`\`

## ⚙️ How to Use
1. Clone or download this repository to your local machine.
2. If this app relies on a third-party weather API (like OpenWeatherMap), ensure you add your specific API key inside the \`index.js\` file.
3. Open the \`index.html\` file in any modern web browser.
4. Type a city name into the search bar and click the search icon to fetch the weather details!
